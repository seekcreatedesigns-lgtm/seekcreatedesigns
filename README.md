# Seek Create Designs Website

This repository contains a ready-to-deploy static website for Seek Create Designs (seekcreatedesigns.ca). 
It uses a warm neutral palette, elegant serif headings, subtle fade/slide animations and separate horizontal gallery pages.

Structure:
/
├── index.html
├── current.html
├── sold.html
├── prints.html
├── commissions.html
├── style.css
├── script.js
├── CNAME
├── README.md
└── /images/
    ├── current/
    ├── sold/
    ├── prints/
    └── commissions/

How to use:
1. Replace placeholder SVGs in the HTML pages with your actual image files located in the appropriate /images/... folder.
2. Push this repo to GitHub and enable Pages (main branch, root).
3. Add the CNAME file (already included) and set DNS A records to GitHub Pages IPs:
   - 185.199.108.153
   - 185.199.109.153
   - 185.199.110.153
   - 185.199.111.153
4. Optionally add a CNAME for 'www' to <yourusername>.github.io
5. Enable HTTPS in repository Pages settings after DNS propagates.
